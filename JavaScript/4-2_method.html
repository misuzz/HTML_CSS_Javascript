Javascriptの書式！

■＜script type="text/javascript"＞＜/script＞の間にコードを書くと実行してくれる。
■headでもbodyでもどっちでも認識してくれるよ。
■でも、「headに書くのは定義部分、bodyに書くのは実行部分」って言われるらしい。

□＜script type="javascript" src="ファイル名.js"＞＜/script＞と書けば、シートを読み込めるよ！
□＜link rel="stylesheet" href="ファイル名.css" type="text/css"＞と書けば、CSSシートの読み込み！

Object

既に用意されてる部品だよ！

  1. 定義済みオブジェクト
    • window　　　　　　　　　　　　　　…　webブラウザそのものを扱う
    • 　　|-document　　　　　　　　　　…　webブラウザの表示領域を扱う
    • 　　　　　|--form　　　　　　　　　…　form作ったときの全体を扱う
    • 　　　　　　　　|---element　　　　…　form作ったときのそれぞれの要素を扱う
    • 　　　　　|--images　　　　　　　　… 画像を扱う
  2. 組み込みオブジェクト
    • Date　　　…日付や時刻を扱う
    • Array　　　…配列にデータを格納する
    • String　　…文字列を扱う

メゾット(動作)とプロパティ（属性）

オブジェクト.メゾット（値）；
オブジェクト.プロパティ　＝　”値”；

「メゾット」って？

■オブジェクトに対する動作だよ。
■ボールペンの「芯を出す」、「文字を書く」
■オブジェクト.メゾット（値）；で指定するよ
■（例）wimdow.document.write("ねちんこちん");

「プロパティ」って？

■各オブジェクトが持つ属性（性質）だよ
■ボールペンの「ケース」はそのボールペンを識別しえる１つの部品（オブジェクト）
■ケースの色は赤（プロパティ）
■（例）window.document.bgColor="red";
　　　　　ボールペン　.　ケースの色= 赤だよ

ここからJavaScript書いてくよ！

  • window.document.write("ほにゃほにゃ");で、表示できるよ
  • window.document.write("＜img src='URL~~'ほにゃほにゃ");でイメージ入れれるよ
  • window.alert="〇〇"で、そのページを開いたときに出てくる警告ポップアップ！
    　window.alertは、「,」で区切ると、区切った前の値だけ出して終わっちゃうよ
    　alertが出てる間は他のjavascriptは止まってるよ
  • window.prompt("説明文","テキストボックスの初期値");で入力するとこ出てくるよ！
  • window.confirm("確認すること");で　「OK(true)」　「キャンセル(false」　出てくるよ！
  • window.document.bgColor="色";で、背景色きめれるよ！
    「＝」だで注意！！！
  • window.document.fgColor="blue";で文字の色決めれるよ！
    「＝」注意！！！
