DATEオブジェクト

getメゾットなので、新しく作ったオブジェクト「new　Date()」（今の時間が入ってるやつ）を使って、今の時間を細かく表示できるよ！

  • getFullYear
  • getMonth※
    ※こいつだけは0~11の数字を返してくるから、表示は 「+1」　！
  • getDate
  • getHours
  • getMinutes
  • getSeconds
  • getDay※
    ※曜日を表示するメゾットだけど、「0~6」で返される。。。
    →Arrayで配列作って、事前に0~6は日〜土ですよ！って入れておく！

Array配列の作り方！

■　var 変数　= new Array(配列の個数);
■　var 変数　= new Array("代入する値を直接入れる","","",…)
　※[]じゃなくて()だよ！呼び出しは　変数[番号]だよ！

var 変数名　= ['あああ', 'いいい','ううう'];　でも配列になるよ！
var 変数名　=　[]; 　でも配列だと思ってくれるよ！
変数名['apple']= 100; と書いて、配列の各箱に名前つけれるよ！（連想配列）
