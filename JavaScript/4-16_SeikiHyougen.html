正規表現

正規表現とは

文字列の検証と置換！
var regex = new RegExp("Apple");
または
var regex = /Apple/;

使い方

var juice = "Apple juice";
var regex = /Apple/;
window.document.write(juice.replace(regex, "Orange"));

文字列が数字だけなのかチェックしたい

var str = "12345678";
var regex = /^[0-9]+$/;
regex.text(str);

■正規表現の例

  • /^[0-9]+$/…数字のみか？
  • /^[a-z]+$/…小文字アルファベットのみか？
  • /^[a-zA-Z]+$/…大文字または小文字のアルファベットのみか？
  • /^[0-9a-zA-Z]+$/…半角英数字のみか？

^…先頭、$…文字の末尾
