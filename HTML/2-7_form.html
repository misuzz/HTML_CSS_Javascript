いいから入力フォーム見たい

部品inputだよ
<input></input>って書くよ
inputはコントロールを作成するよ！

  • input type="text"
    ->１行入力テキストボックス！
  • input type="text" size="20"
    ->半角で２０文字表示するよ
  • input type="text" maxlength="30"
    ->入力可能な文字数は半角で３０文字ですよ
  • input type="radio"
    ->ぽち
  • input type="type" checked="true"
    ->初期状態からポチが入ってるよ
  • input type="checkbox"
    ->radioとちがって複数チェックできるよ
  • input type="~~" name="なんちゃら"
    ->id属性と同じで、その場所に名前をつけるよ
    ->これつけてあげないと、ラジオボタン作っても全部に点つけれちゃうよ
    これで同じカテゴリだよーってやってあげると選択式になるよ！
  • input type="~~" value="なんちゃら"
    ->textならテキストの中に「なんちゃら」って入るし、
    ->submitボタンであればボタンに「なんちゃら」って入るよ

  • select name="なんちゃら"
