スタイル規則の書式

  1. セクレタ{プロパティ：値；}

    2. p{color:red:}みたいな

  2. セクレタ{プロパティ:値;プロパティ：値；....}

    3. p{color:red;font-size:large;}みたいな

  3. セクレタ,セクレタ,セクレタ{プロパティ：値；}

    4. h1,h2,h3{color:red;}みたいな

子孫セクレタ！

例えば、　<p><b>んわんわ</b>おうおう</p>　だとすると
ｂは子、ｐは親。
CSSで
p b{color:red;}って書くと、
「親がｐで子がｂのところは赤字になるよー」ってなる

継承！

pの中にspan書いた時、
pで指定したものもspanに反映されるよ！

あーここんとこだけ色変えたいんだけどどうやって指定しよかな、、

htmlのタグに「class="〇〇"」つけてくれればCSSで指定できるよ！
例：<p class="red">赤にしたいよ<p>

このときのCSSの書き方２選

  1. p.red{color:red;}
  2. .red{color:red;}

もう既に決められてるクラス名

もう決まってるクラスを使うときは.クラス名じゃなくて「:クラス名」

  1. a:link{color:blue;}

    2. リンクになってるところの色を変える

  2. a:visited{coloer:red;}

    3. 既にアクセスしたリンクの色を変えれる

  3. a:hover{color:red;}

    4. マウスを載せた時の色を変えれる

  4. a:active{color:red:}

    5. クリックした瞬間の色を変えれる

IDセクレタ

ページの中で１回しか使えないクラス！
例：htmlで div id="header"としたら、
CSSで　#header{background-color:red;} と書く
